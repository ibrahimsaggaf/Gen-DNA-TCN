# Usage
This repository contains the implementation of Gen-DNA-TCN. The implementation is built in Python3 (version 3.10.12) using Scikit-learn, Levenshtein, and the deep learning library Pytorch.

## Requirements
- torch==2.1.2
- scikit-learn==1.4.0
- Levenshtein==0.26.1
- pandas==2.2.3
- numpy==1.26.4
